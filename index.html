Este módulo é independente do Restaurar. Gera um ficheiro .sh para executar no Ubuntu/Debian com sudo: instala o servidor (pacote do repositório), cria o banco de destino se não existir e cria o utilizador com as mesmas credenciais da conexão VaultDB escolhida (para o restore posterior funcionar sem ajustes manuais de permissões).
